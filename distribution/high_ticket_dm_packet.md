# HIGH-TICKET DM PACKET

## For Direct Outreach to Premium Prospects

Use this for targeted DM outreach to high-value contacts.

---

## DM SEQUENCE

### Message 1: The Hook

**For cold but qualified:**
```
[Name],

One quick question: is your operations setup costing you revenue right now?

I'm running a high-ticket implementation sprint today and have one slot open.

If you're serious about [automating/scaling/breaking through], I'll show you what's possible.

5 minutes. No pitch.

Reply "YES" if interested.
```

**For warm contacts:**
```
[Name],

Quick — I have one premium implementation slot available today.

AI Revenue Engine Sprint. 48 hours. £15K.

If you're ready to [transform your operations / automate revenue], let's talk.

Reply "IN" and I'll send details.
```

---

### Message 2: Follow-Up (2 hours later, no response)

```
Changing topics — but if you're ever looking to systemize your business, I built a premium offering that solves the exact problem you're probably facing.

No hard sell. Just a resource if you need it.

Enjoy your [day/week].
```

---

### Message 3: Final (End of day, no response)

```
Last check — I'm closing out the day with one premium spot still open.

If you want in, now's the time.

Otherwise, I'll reach out again [next week/month].

Best,
```

## SAME-DAY CLOSE SCRIPTS

### For "Tell me more"

"Here's the quick version — I help [target] add £10K+/month through AI automation. The implementation sprint is £15K and delivers in 48 hours. Want to see if it's a fit?"

### For "What's the investment?"

"£15K for the sprint. That includes the full AI system, 30-day support, and training. You saw my results with [proof point] — same framework applied to your business."

### For "Can you do it for less?"

"What would make this a priority? Let's find a path that works. I also have a £2,500 audit option if the sprint isn't the right fit."

## CLOSE TRIGGERS

When they respond positively:
1. Ask for a quick call (within 30 mins)
2. On the call: qualify → present → close
3. Get deposit BEFORE hanging up
4. If they hesitate: use objection handling script
