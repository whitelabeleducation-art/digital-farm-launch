# DAY 0 ENGAGEMENT MAP

## Signal Response Matrix

### TYPE A: Comment "How do I get this?"
**Time to respond:** < 15 minutes
**Response:** "Link is in the post! Takes 30 seconds. No email required."
**Log:** COMMENT → signal +1

---

### TYPE B: Comment "Is this free?"
**Time to respond:** < 15 minutes
**Response:** "Yes, free for 48 hours to validate demand. After that $47."
**Log:** COMMENT → signal +1

---

### TYPE C: Comment "What tools does it work with?"
**Time to respond:** < 30 minutes
**Response:** "ChatGPT, Claude, and Zapier. All work with free tiers - no paid subscriptions."
**Log:** COMMENT → signal +1

---

### TYPE D: DM "interested" / "tell me more"
**Time to respond:** < 5 minutes
**Response:** "Which workflow are you most excited about? (Research, Content, Outreach, Data, Support?)"
**Log:** DM → signal +2 → then follow-up

"Awesome! Here's the payment link: [PAYLINK]. Founding price is $37 (regular $47). Use code FOUNDING20."

**Response:** "Got it! Here's your bundle: [DOWNLOAD LINK]. Quick start: try the Research Agent first."
**Log:** SALE → signal +10

---

### TYPE E: DM "I want to buy"
**Time to respond:** < 2 minutes

**After payment:**
"Here's your download link: [DOWNLOAD LINK]. Let me know if you have questions!"

**Time to respond:** < 5 minutes

## Escalation Tree

### If no response within 4 hours (TYPE D, E, F):
- DM: "Hey - just checking if you got the link working?"

### If no response within 24 hours (any type):
- DM: "Hey - not trying to be pushy, but free offer expires in [time]. Let me know if you want in!"

### If sale requested but no payment within 2 hours:
- DM: "Just a reminder - the payment link is [PAYLINK]. Let me know if you need anything!"
