# VALIDATION SCORECARD - Real-Time

## Scoring Rules

| Signal | Weight | Description |
|--------|--------|-------------|
| DM with question | 1 | Someone reached out with real question |
| DM "interested" | 2 | Explicit interest expressed |
| DM "tell me more" | 2 | Active interest, needs nurturing |
| Comment asking for link | 2 | Public display of interest |
| Comment "how does this work" | 2 | Considering use |
| Email captured | 2 | Gave contact info |
| Reply "WORKFLOWS" | 3 | Action on CTA |
| Asked about price | 3 | High intent |
| Asked about buying | 5 | Ready to buy |
| Preorder requested | 7 | Committed intent |
| Paid sale | 10 | REAL REVENUE |
| Shared post | 4 | Endorsed without asking |
| Tag/mention | 3 | Social proof |

### HOT (5-7 points)

### CLOSED (10 points)
