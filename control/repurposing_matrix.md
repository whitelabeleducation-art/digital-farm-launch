# Repurposing Matrix

## Content Atomization

One piece of content → Multiple formats → Multiple channels

### Original: "I replaced my VA with AI agents"

| Format | Platform | Variant |
|--------|----------|---------|
| Long post | Reddit | Full story, all details |
| Thread | X | 5-tweet breakdown |
| Short post | X | One hook + CTA |
| LinkedIn | LinkedIn | Business-focused, ROI |
| Quote graphic | All | Key stat |
| Video (future) | YouTube/TikTok | Walkthrough |

---

## Content Translation Rules

### Reddit → X
- Remove paragraphs
- Add hooks
- Truncate to fit
- Add CTAs

### X → LinkedIn
- Expand context
- Add professional framing
- Remove Twitter-isms

### Blog → All
- Key points → Twitter thread
- Stats → Infographic
- How-to → Video script
