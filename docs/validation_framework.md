# Validation Framework - Product Validation Process

## Validation Stages

### Stage 1: Demand Discovery
- **Method**: Mine existing signals (reddit, search, communities)
- **Success Criteria**: 5+ clear pain point mentions in 48 hours
- **Output**: Confirmed demand signal

### Stage 2: Audience Definition
- **Method**: Define specific customer avatar
- **Success Criteria**: Clear demographics + psychographics
- **Output**: Audience profile

### Stage 3: Problem Definition
- **Method**: Articulate the specific painful problem
- **Success Criteria**: One sentence problem statement
- **Output**: Problem statement

### Stage 4: Transformation Definition
- **Method**: Define the before/after state
- **Success Criteria**: Clear "this to that" statement
- **Output**: Transformation promise

### Stage 5: Format Definition
- **Method**: Determine best delivery format
- **Success Criteria**: Clear format (template, guide, tool, etc.)
- **Output**: Product format

### Stage 6: Price Hypothesis
- **Method**: Research competitor pricing + value based
- **Success Criteria**: Price point with reasoning
- **Output**: Price hypothesis

### Stage 7: Free Validation
- **Method**: Pre-sell, waitlist, lead magnet signup, DM outreach
- **Success Criteria**: 10+ interested signals
- **Output**: Validation evidence

### Stage 8: Kill/Launch Decision
- **Method**: Review all evidence vs criteria
- **Kill Criteria**: 
  - Score < 5
  - No demand signal after 2 weeks
  - Unable to differentiate
- **Launch Criteria**:
  - Score > 6
  - Clear demand signal
  - Clear differentiation
  - Clear path to audience

## Common Validation Methods (Free)

### Quick Validations (Hours)
- Reddit post with "would you buy?" check
- X/Twitter poll
- LinkedIn post with problem framing
- DM to 10 potential customers
- Comment extraction from existing posts

### Medium Validations (Days)
- Waitlist landing page
- Lead magnet signup page
- Pre-order form ( Gumroad ready)
- Content series testing response

### Slow Validations (Weeks)
- Actual product launch
- Beta program
- Paid ads (only after validation)
