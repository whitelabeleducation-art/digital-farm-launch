## PROOF ASSET TYPES

### 1. Case Studies
- Problem → Solution → Result format
- 300-500 words
- Include metrics

### 2. Before/After Comparisons
- Visual or tabular
- Specific metrics (time, revenue, efficiency)

### 3. Diagnostic Tools
- Quiz format
- Score results with recommendation

### 4. Implementation Guides
- Step-by-step
- Timeline estimate
- Investment guide

### 5. Results Frameworks
- ROI calculator template
- Impact estimation
